-------------------------------------------
$ cd examples
$ python run_market_research.py

Similar to Method 1 but with less detailed formatting.
Still generates 10+ business ideas for your industry.


METHOD 3: PROGRAMMATIC (For advanced users/integration)
-------------------------------------------------------
import asyncio
from orchestrator.execution_engine import AgentOrchestrator

async def discover():
    orchestrator = AgentOrchestrator()
    results = await orchestrator._run_industry_research("YourIndustry")
    
    for idea in results['business_ideas']:
        print(f"💡 {idea['name']}: {idea['overall_opportunity_score']}/10")

asyncio.run(discover())

⚡ PRO TIPS
================================================================================

TIP 1: Get Different Results
Run the tool multiple times with different focus areas:
  - "HealthTech" (broad)
  - "HealthTech - Mental Health" (specific)
  - "HealthTech - Preventive Care" (different angle)

TIP 2: Compare Industries
Run for multiple industries to find which has better opportunities:
  $ python discover_business_ideas.py  # Run for HealthTech
  $ python discover_business_ideas.py  # Run for FinTech
  $ python discover_business_ideas.py  # Run for ClimaTech

TIP 3: Filter Ideas
Look at the JSON report and filter by criteria:
  - Overall score >= 8 (high opportunity)
  - Competition level == "low" (less crowded)
  - Time to market <= 6 months (quick launch)

TIP 4: Validate Top Ideas
For your favorite 3 ideas:
  1. Interview 20+ potential customers
  2. Ask if they'd pay for this solution
  3. Research existing competitors
  4. Test core assumptions

TIP 5: Use with Your Team
  - Show top 3 ideas to co-founders
  - Discuss feasibility and interest
  - Pick one to explore further
  - Create detailed business plan

================================================================================
🆘 TROUBLESHOOTING
================================================================================

ISSUE: Tool doesn't start
SOLUTION:
  - Ensure Python 3.8+ is installed
  - Check all dependencies: pip install -r requirements.txt
  - Verify you're in the correct directory

ISSUE: "Ideas look too generic"
SOLUTION:
  - Provide more specific industry focus
  - Run multiple times (each run generates different ideas)
  - Try niche or emerging industries
  - Check the JSON report for more detailed analysis

ISSUE: API timeouts or errors
SOLUTION:
  - Check internet connection
  - Verify API credentials
  - Wait and retry (API may be rate-limited)
  - Check API service status

ISSUE: Reports directory not created
SOLUTION:
  - Verify write permissions in project directory
  - Try creating directory manually: mkdir reports
  - Run with administrative privileges if needed

ISSUE: Can't find the JSON report
SOLUTION:
  - Check the reports/ directory in the project root
  - Look for files named business_ideas_*.json
  - Sort by date modified to find the latest
